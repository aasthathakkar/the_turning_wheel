# Product Photo Manifest

Drop photos into these folders using these exact filenames. Nothing else needs to change — the site will pick them up automatically. Missing files just show the placeholder, so you can add these gradually in any order.

Accepted format: **.jpg** (save/export your photos as .jpg). Recommended size: roughly 1200×900px or similar 4:3-ish ratio, under ~500KB each so the site stays fast.
